# Word-Clock

    DIY Word Clock ver. 1.01
    
    Used in the project:
    Arduino nano
    RTC DS3231
    SK6812 RGBW 30 IP20 Strip
    Power supply MeanWell PM-15 5V 15W 
    
    Arduino library:
    <iarduino_RTC.h>
    <iarduino_RTC_DS3231.h>
    <Adafruit_NeoPixel.h>
